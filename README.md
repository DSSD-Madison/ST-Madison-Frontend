# Strong Towns DSSD Frontend

This is the frontend for the Strong Towns DSSD project. It is built with SvelteKit and Vite.

## How to Run

You will need to run the backend server too! You can follow the instructions in the [backend README](https://github.com/DSSD-Madison/ST-Madison-Backend) to get that up and running.

```bash
pnpm install
pnpm dev
```
